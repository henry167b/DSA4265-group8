# Financial Investor Note Generator

An AI-powered agent that automatically sources SEC filings (10-K/10-Q forms), financial documents, and P&L statements from selected companies, then generates concise investor notes.

## Overview

This tool streamlines the equity research workflow by:

1. **Retrieving SEC Filings** - Automatically fetches 10-K (annual) and 10-Q (quarterly) reports from the SEC EDGAR database for specified companies.
2. **Extracting Financial Data** - Parses key financial statements including income statements, balance sheets, and cash flow data.
3. **Generating Investor Notes** - Uses an LLM to synthesize the extracted data into structured investor notes covering financial highlights, risk factors, and key takeaways.
